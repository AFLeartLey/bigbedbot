## Configuration

根据自身情况以及 config_example.json 和 group_permission_example.json 创建对应的文件
## Launch

  1. 使用 `poetry` 以及 `python 3.8` 安装对应的库
  
  2. 打开 `mcl` （版本需高于2.0）
  
  3. `poetry run python3 main.py`
